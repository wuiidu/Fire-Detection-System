**🚀 Hướng dẫn cài đặt và khởi chạy**

Dự án đã được tự động hóa hoàn toàn quy trình thiết lập môi trường ảo và cài đặt thư viện cần thiết.



1\. Tải (Clone) repository này về máy.

2\. Chạy file KhoiDong.bat (nhấn đúp chuột). Script sẽ tự động:

&#x20; - Khởi tạo môi trường ảo Python.

&#x20; - Cài đặt các thư viện cần thiết từ file ThuVien.txt.

&#x20; - Khởi chạy ứng dụng chính.
